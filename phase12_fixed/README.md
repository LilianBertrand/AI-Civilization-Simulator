# AI Civilization Simulator

A Python and Streamlit multi-agent simulation engine where civilizations grow, trade, fight, industrialize and generate their own strategic history.

## Highlights

- Random world generation with seed replayability
- Strategic map filters: Territory, Terrain, Resources, War, Religion, Economy and Technology
- Dynamic economy with resources, supply/demand and commodity prices
- Diplomacy, wars, religions, technologies, leaders, crises and modern-world systems
- Timeline Intelligence with event explanations
- Civilization Comparator for country-style analysis
- Analytics Lab with volatility, correlations, rankings and export tools
- Final Simulation Report exportable as Markdown

## Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly

## Run locally

```bash
pip3 install -r requirements.txt
streamlit run app.py
```

## Suggested use

1. Choose a scenario and volatility mode.
2. Generate a new random world or enter a fixed seed.
3. Run 500+ simulated years.
4. Explore the Command Center, Strategic Map Room, Timeline Intelligence and Analytics Lab.
5. Export the final simulation report for documentation.
